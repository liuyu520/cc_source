# `src/components/tasks/` 模块索引

## 模块定位

`src/components/tasks/` 负责后台任务与远程会话的展示层，包括任务列表、详情弹窗、进度视图和状态工具函数。

- `BackgroundTask.tsx`

- `BackgroundTaskStatus.tsx`
  状态展示
