# `src/services/autoEvolve/learners/` 模块索引

## 模块定位

`learners/` 是自演化中的学习器层，负责学习器运行时、hook gate、技能路由与 prompt 片段管理。

## 文件清单

- `runtime.ts`
- `shared.ts`
- `hookGate.ts`
- `skillRoute.ts`
- `promptSnippet.ts`
- `autoContinue.ts`

## 作用概括

- `runtime.ts`
  学习器运行时主逻辑
- `hookGate.ts`
  学习器与 hook 的闸门
- `skillRoute.ts`
  技能路由
- `promptSnippet.ts`
  prompt 片段资产
- `autoContinue.ts`
  与自动续写/自动继续链路的接口
